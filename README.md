# Sistem Pengenalan Bahasa Isyarat Indonesia (BISINDO)

Proyek tugas akhir / skripsi ini berfokus pada pengembangan sistem klasifikasi dan pengenalan abjad Bahasa Isyarat Indonesia (BISINDO) secara *real-time* berbasis *Computer Vision* dan *Deep Learning*.

Sistem ini dirancang untuk mendeteksi serta mengklasifikasikan gestur tangan abjad BISINDO menggunakan webcam secara interaktif. Pemrosesan data gestur dilakukan di bagian *backend* menggunakan Python, sementara antarmuka pengguna berbasis web (*frontend*) digunakan untuk menampilkan hasil deteksi secara *real-time*.

------------------------------------------------------------------------------

##Menjalankan Backend
1. Install seluruh kebutuhan dependencies pada requirements.txt
2. Jalankan aplikasi server app.py

##Menjalankan Frontend
1. Pastikan server sudah berjalan terlebih dahulu
2. Buka folder frontend di Visual Studio Code (VS Code)
3. Klik kanan pada isi file index.html, lalu pilih "Open with Live Server"
